Enterprise Authentication

Methods:
JSON Web Token, oAuth, Bearer Token, Esri Token, Basic Authentication, Key Files, Encryption/Decryption
Systems:
Esri Enterprise, Online, Services, GDB, RDBMS, Oracle Utilities
Languages and Technologies:
Java, JavaFX, Eclipse, Maven, Postman, Jose4j, Java Security, License3j

Purpose:
To create a unified authentication system that supports multiple protocols and integrates with enterprise systems, ensuring both secure access and robust data integrity

Business Need:
With a growing mix of legacy and cloud-based applications, the organization needed a centralized, secure method to manage user authentication and protect sensitive data

About This Project

As a Senior Integration Engineer, I spearheaded the development of an Enterprise Authentication & Security Framework that unifies and secures access across a diverse range of legacy and modern applications. In this project, I integrated multiple authentication protocols—including JWT, OAuth 2.0, and Oracle Advanced Authentication—by designing custom middleware that intelligently determines and routes authentication requests. This approach effectively bridged the gap between legacy SOAP-based services and modern RESTful APIs, ensuring a consistent and secure user experience across all platforms.

I also implemented a robust token management system which significantly enhanced performance and reduced the risk of unauthorized access. In parallel, I architected an integrated product licensing suite that automates secure license file generation. By employing advanced encryption techniques like AES and RSA, I ensured that sensitive data remains protected while maintaining compliance with stringent industry standards.

Throughout this initiative, I combined a deep understanding of secure software design with a commitment to scalability and operational efficiency. My work not only improved our overall security posture but also streamlined system integrations, reduced authentication failures, and enhanced the user experience. This project stands as a testament to my ability to seamlessly integrate complex security measures into enterprise software, delivering a solution that is both resilient and future-proof.
